# 🚀 Ram-Optimizer Pro

**Ram-Optimizer Pro** is a lightweight, aggressive system optimization suite designed for legacy hardware and 32-bit systems. It combines RAM management with active CPU temperature control to keep your vintage hardware running smoothly.

## 🔥 Key Features

- **❄️ S-Direct Cooler:** Instantly drops temperatures by throttling the CPU and killing non-essential background tasks.
- **⚡ Turbo RAM Purge:** Forces RAM usage from ~70% down to ~40% by refreshing Swap and dropping caches.
- **🛡️ Auto-Guard:** Background monitor that triggers cooling and purging automatically (75°C or 70% RAM).
- **🧹 System Vacuum:** Cleans journals and APT caches to free up disk space.
